# ToDoList
A simple html webpage with CSS and Javascript functions to create and manage tasks.
Created by Les Malan, 2026. All rights reserved.

Design criteria:
    Webpage that can display to do list in a browser
    List of incomplete tasks
    List of completed tasks
    Method to move tasks from incomplete to complete and vice versa
    Method to delete tasks
    Method to add new tasks

Webpage Structure:
    #Main page heading
    #Sub-heading for incomplete tasks - 
    #Area to display incomplete tasks - 
    #Sub-heading for completed tasks - 
    #Area to display completed tasks - 
    #Area to add new tasks - 

Interactive Parts:
    #List objects to hold input for each list - 
    #Check mark button on each task to move from incomplete to complete list object - 
    #Cross button to delete incomplete tasks - 
        Confirmation dialog box
    #Clear check mark button on completed tasks to move back to incomplete - 
    #Text entry box to name new tasks - 
    #Button to commit new tasks to incomplete list

Formatting:
    #Lists surrounded by border box
    #Incomplete tasks are bolder shade and larger font
    #Completed tasks have strike-thru formatting and light shade
    #Sub-headings larger font than lists
    Color, background, element spacing
